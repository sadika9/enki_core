# Installation

Install `enki-py` from PyPI.

## Requirements

- Python `>=3.8`
- `pip` or `uv`

If you use `pip`:

```bash
pip install enki-py
```

If you use `uv`:

```bash
uv add enki-py
```

## Quick check

```bash
python -c "from enki_py import Agent; print(Agent)"
```

For package metadata and releases, see https://pypi.org/project/enki-py/
